## 0.1.0 - 2026-06-19

### Added

- Initial local Paper/Purpur 26.2 plugin baseline.
- Server-side tree chopping when breaking natural trees with an axe.
- Connected log and wood-family detection with nearby natural leaf checks.
- Scheduled block breaking to limit tick spikes and optional axe durability consumption.
- Natural leaf removal so vanilla drops still apply.
- `/treecapitator` command surface with status, reload, selftest, and help commands.
- Local Java 25 Gradle build and `server-dev` validation workflow.
